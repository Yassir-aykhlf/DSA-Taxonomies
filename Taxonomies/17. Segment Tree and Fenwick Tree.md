# Segment Tree & Fenwick Tree Pattern Taxonomy

> Tree-backed arrays that answer range queries and apply updates in O(log n), where a static prefix-sum array falls short because the values keep changing.

```text
└── Segment Tree & Fenwick Tree Pattern
    │
    ├── Segment Tree — Point Update, Range Query
    │   │
    │   └── Mutable Range Sum / Min / Max

    │       ├── "307. Range Sum Query - Mutable"
    │       └── "327. Count of Range Sum"
    │
    ├── Segment Tree — Lazy Range Update
    │   │
    │   └── Defer Range Writes to Children
    │       │
    │       ├── "699. Falling Squares"
    │       ├── "370. Range Addition"
    │       └── "715. Range Module"
    │
    ├── Coordinate-Compressed Interval Sweep
    │   │
    │   └── Skyline / Calendar / Rectangle Area
    │       │
    │       ├── "218. The Skyline Problem"
    │       ├── "732. My Calendar III"
    │       ├── "850. Rectangle Area II"

    │

        │
        └── Threshold Square Side / Block Sum
            │
```

> **Related Patterns**:
> - Prefix Sum (the simpler choice when the array never changes)
> - Binary Search (drives Fenwick-tree order-statistic queries)
> - Divide & Conquer (segment-tree construction is a recursive split)
> - Sorting (coordinate compression sorts endpoints before the sweep)
