> **Related Patterns**:
> - Dynamic Programming (bitmask DP encodes subsets and visited states as ints)
> - Math (many arithmetic operations have faster bitwise forms)
> - Trie (a binary trie solves maximum-XOR bit by bit)
> - Hash Table (bitmasks act as compact, hashable set keys)
