> **Related Patterns**:
> - Two Pointers (a space-free alternative for duplicate and pair checks)
> - Sliding Window (pairs with a count map to track window frequency)
> - Prefix Sum (a prefix-count map turns subarray sums into O(n) lookups)
> - Trie (an alternative to hashed signatures for prefix-heavy strings)
