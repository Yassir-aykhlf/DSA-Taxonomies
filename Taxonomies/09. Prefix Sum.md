> **Related Patterns**:
> - Sliding Window (an alternative for some contiguous range-sum problems)
> - Hash Table (a prefix-count map counts subarrays in a single pass)
> - Segment Tree (handles the mutable-array case prefix sums cannot)
> - Dynamic Programming (prefix aggregates speed up many transitions)
